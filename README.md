# Campus Navigator Mobile App

## Project Overview

Campus Navigator is a mobile application prototype designed to help students, staff, and visitors easily navigate the university campus. The application enables users to search for buildings, locate lecture halls, laboratories, administrative offices, and essential campus services through a simple and user-friendly interface.

This project is developed as part of the Human-Computer Interaction (HCI) Mobile Application Group Assignment.

---

## Problem Statement

New students and visitors often experience difficulties locating buildings, lecture halls, offices, and university services on campus. Existing methods such as printed maps or asking for directions are inefficient and time-consuming. This mobile application aims to provide a simple digital solution for campus navigation.

---

## Objectives

- Design a user-friendly mobile navigation application.
- Enable users to search for campus buildings and services.
- Provide simple navigation directions.
- Allow users to save frequently visited locations.
- Demonstrate Human-Computer Interaction principles through an interactive prototype.

---
